## Getting Started

1. Download and install PIDToolBox on your system.  
2. Launch the application and load a blackbox log from your flight controller.  
3. Select the spectral or step-response tool and configure the view.  
4. Analyze the data, adjust your PID/filters based on results, then retest on air.
